Distribution Plot

Distribusi Tingkat Literasi Negara di Dunia

What?

Visualisasi Distribution Plot yang dibangun merepresentasikan distribusi tingkat literasi di dunia.

When?

Data yang ditampilkan adalah data tingkat literasi di dunia pada tahun 2017.

Why?

Distribution Plot digunakan karena baik dalam merepresantikan persebaran suatu data.

How?

Distribution Plot dapat dibangun dengan menggunakan library Plotly Figure Factor dengan memanggil fungsi “create_distplot”


import plotly.figure_factory as ff
#Distribution Plot
x = df['Literacy (%)']
hist_data = [x]
group_labels = ['Literacy (%)'] # name of the dataset
fig = ff.create_distplot(hist_data, group_labels, show_rug=False, curve_type='kde', bin_size=5)
fig.update_layout(title_text='Distribusi Tingkat Literasi Negara di Dunia')
fig.show()
