Tentang Dataset

Dataset yang digunakan adalah “Countries of The World” yang berisikan data-data seperti luas area populasi, kepadatan penduduk, jumlah migrasi, tingkat literasi dan GDP yang dikategorikan per negara.

Link Dataset https://www.kaggle.com/fernandol/countries-of-the-world
Jumlah Kolom 20
Jumlah Baris 227
Tipe Data Tabular

Data Pre-processing

Karena dataset masih memiliki missing value, maka perlu dilakukan penindakan lebih lanjut untuk data tersebut. Langkah yang dipilih adalah untuk mengganti null value pada dataset dengan nilai rata-rata (mean).
