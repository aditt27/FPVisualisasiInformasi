Map

Tingkat Migrasi Tiap Negara

What?

Map tersebut memvisualisasikan tingkat migrasi pada tiap negara. Dari visualisasi tersebut, didapatkan informasi mengenai negara-negara dengan tingkat migrasi tinggi yang direpresentasikan dengan warna hijau dan tingkat migrasi rendah yang direpresentasikan dengan warna merah.

When?

Data yang ditampilkan adalah data tingkat migrasi tiap negara pada tahun 2017.

Why?

Visualisasi Map digunakan agar dapat merepresentasikan data dengan level per negara dengan baik, sehingga pengguna informasi dapat melihat informasi tersebut sesuai dengan lokasi letak geografis pada masing-masing negara.

How?

Visualisasi dibuat dengan menggunakan library Plotly Graph Objects agar visualisasi lebih interaktif. Dalam library tersebut, map dibangun dengan menggunakan fungsi “Choropleth”. 


import plotly.graph_objects as go
#Map
fig = go.Figure(data=go.Choropleth(
        locationmode = 'country names',
        locations = df['Country'],
        z = df["Net migration"],
        text = df['Country'],
        autocolorscale = True,
        reversescale = False,
        marker = dict(
            line = dict (
                color = 'rgb(180,180,180)',
                width = 0.5
            ) )
        ))
fig.update_layout(
    autosize=True,
    title_text='Tingkat Migrasi Tiap Negara',
    geo=dict(
        showframe = True,
        showcoastlines = True,
        showocean=True,
        projection = dict(
            type = 'orthographic'
        )
    )
)
fig.show()
