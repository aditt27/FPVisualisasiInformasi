Scatter Plot

Hubungan Antara Tingkat Literasi Dengan Kematian Mayi

What?

Scatterplot akan merepresentasikan hubungan antar dua variabel, dalam hal ini adalah hubungan antara tingkat literasi dan angka kematian bayi.

When?

Data yang ditampilkan adalah data tingkat literasi dan kematian bayi tiap negara pada Region tertentu tahun 2017.

Why?

Scatter Plot dipilih agar dapat memvisualisasikan hubungan antar variabel secara baik.

How?

Visualisasi dibuat dengan menggunakan library Plotly Express agar visualisasi lebih interaktif. Dalam library tersebut, map dibangun dengan menggunakan fungsi “scatter”.


import plotly.express as px
#Scatter Plot
fig = px.scatter(df, x="Infant mortality (per 1000 births)", y="Literacy (%)", color="Region",
                 hover_data=['Country'], title = "Hubungan antara tingkat literasi dengan kematian bayi")
fig.show()
